# requirements
* [python2 and pip](https://www.python.org/)
* [GAE SDK for Python](https://cloud.google.com/appengine/downloads)
* [Bower](http://bower.io/)

# how to use
1. Download and unarchive on the project root
1. Run ```pip install -t lib -r requirement.txt``` (required only once)
1. Run ```bower install``` (required only once)
1. Run ```dev_appserver.py .```
1. Visit ```(host)/init``` on your browser and log in as Administrator. (required only once)
1. Now, it's ready to use.
